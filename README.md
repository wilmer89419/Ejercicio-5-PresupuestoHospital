# PresupuestoHospital
: Crear un programa que lea el presupuesto de un hospital y se distribuya en
diferentes departamentos según el porcentaje asignado, el cual se presenta a
continuación:

Departamento Porcentaje

Emergencia 30%
Cirugía 15%
Suministros 25%
Administración 18%
Laboratorio 5%
Imágenes Diagnosticas 7%
